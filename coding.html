Welcome To

Principles Of Programming

Summer School is a cumulative initiative of all the prestige technical institutions of the country including IITs,NITs, IIITs and GFTIs for setting up an environment and teach them Introduction to Programming. All the websites, applications and complex algorithms are done with the help of Programming.

WHAT YOU'LL LEARN ?
  • Learn to program with one of the most powerful programming languages that exists today, C++.
  • Obtain the key concepts of programming that will also apply to other programmibg languages
  • Learn Modern C++ rather than an absolete version of C++ that most other courses teach.
  • Learn C++ features from basic to advanced such as inheritance and polymorphic functions.
WHAT THIS COURSE IS FOR:
  • This course is perfect for absolute begineers with no previous coding experience, or anyone wanting to add C++ to their existing skillset.
  • Anyone looking to increase career options by learning one the most in-demand programming languages.
WHAT YOU'LL GET...
  • The most important thing you'll achieve is a skillset of programming via which you can take a direction in various technical fields like App and Web development, ML and AI, Competitive Coding and also Cyber Security.
  • You'll get a certificate for attending this prestigious sessionn and inculcating coding skills from Indian Institute of Information Technology, Lucknow an Institute of National importance.
